JHSawatzki/rpi-custom-kernel
==========

Linux kernel release 3.12.25+ with builtin IPv6 support for the Raspberry Pi.

Install
-------

```text
sudo REPO_URI=https://github.com/JHSawatzki/rpi-custom-kernel rpi-update
```



Changelog
---------
No changes, first release.

-------
